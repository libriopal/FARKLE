SECTION 2 — NEW ACTION TYPES
════════════════════════════════════════════════

ADD these action types to the GameAction union:

WILD_SCATTER:
  payload: { destroyedFaces: DieFace[] }
  REDUCER LOGIC:
    score = wildScatterScore(destroyedFaces)
    newGrid = apply EMPTY to all destroyed cells (already done in applyWildScatter)
    unbanked += score
    wildCount = 0
    globalEnergyMode stays — Frenzy was already triggered upstream
    Add ScorePopup: "WILD SCATTER! +{score}"
    Run cascade after

ROUND_END:
  payload: { reason: RoundEndReason }
  REDUCER LOGIC:
    bank all unbanked → banked (auto-bank)
    multiplierStep = 0
    consecutiveChains = 0
    roundPhase = 'ROUND_END'
    lastRoundEndReason = reason
    roundNumber++ (after board reset in next action)
    If heistState: apply vault split to banked points:
      vaultContribution = banked * VAULT_SPLIT
      personalContribution = banked - vaultContribution
      heistState.vault += vaultContribution
      heistState.personalScores[localPlayerId] += personalContribution
      banked = personalContribution (only personal portion in "banked" display)

ROUND_RESET:
  payload: none
  REDUCER LOGIC:
    Create fresh grid via createGrid() with SixPoolManager
    roundPhase = 'ROLLING'
    Soft-reset: keep banked score, vault, heistState
    Reset: unbanked = 0, multiplierStep = 0, wildCount = 0

HEIST_INITIATE:
  payload: { initiatorId: string }
  REDUCER LOGIC:
    Validate: heistState !== null
    Validate: heistState.betrayalUnlocked[initiatorId] === true
    Validate: localEnergy >= HEIST_ENERGY_COST (checked before dispatch)
    Set heistState.activeHeist = {
      initiatorId,
      countdownMs: HEIST_WINDOW_MS,
      blockerIds: [],
      startedAt: Date.now()
    }
    roundPhase = 'HEIST_WINDOW'

HEIST_BLOCK:
  payload: { blockerId: string }
  REDUCER LOGIC:
    If heistState?.activeHeist:
      Add blockerId to activeHeist.blockerIds

HEIST_RESOLVE:
  payload: { success: boolean }
  REDUCER LOGIC:
    If success:
      stealAmount = heistState.vault
      If Archivist role in session:
        stealAmount = vault * (1 - ARCHIVIST_VAULT_SHIELD)
        remaining = vault * ARCHIVIST_VAULT_SHIELD (stays in vault)
      banked += stealAmount
      heistState.vault = remaining (or 0)
      heistState.heistCount++
      heistState.successfulHeisters.push(initiatorId)
      heistState.activeHeist = null
      roundPhase = 'ROLLING'
    If not success:
      heistState.activeHeist = null
      roundPhase = 'ROLLING'

TICK_HEIST_WINDOW:
  payload: { deltaMs: number }
  REDUCER LOGIC:
    If heistState?.activeHeist:
      activeHeist.countdownMs -= deltaMs
      If countdownMs <= 0: dispatch HEIST_RESOLVE({ success: true })

VAULT_BETRAYAL_CHECK:
  payload: { playerId: string; energy: number }
  REDUCER LOGIC:
    If heistState:
      unlocked = heistState.vault >= VAULT_THRESHOLD
                 && energy >= HEIST_ENERGY_COST
      heistState.betrayalUnlocked[playerId] = unlocked

UPDATE COMMIT_CHAIN (major changes):

  After scoring:
    1. Call applyChainEnergy via useEnergy (passed as callback)
    2. If energy would hit 0 → dispatch ROUND_END({ reason: 'ENERGY_ZERO' })
    3. If score > 0 && chain.length < 6:
         dispatch ROUND_END({ reason: 'SHORT_CHAIN' })
         (auto-bank and end round)
    4. If score > 0 && chain.length === 6:
         continue round (roundPhase stays 'ROLLING')
    5. If score === 0 (Farkle):
         dispatch ROUND_END({ reason: 'FARKLE' })
         NOTE: Farkle during FRENZY still ends the player's round
               but they remain on board in multiplayer
    6. After cascade: call scanForWilds(grid)
       Update state.wildCount
       If wildCount >= WILD_BLOCKER_FRENZY_COUNT:
         const { newGrid, destroyedFaces } = applyWildScatter(grid)
         grid = newGrid
         dispatch WILD_SCATTER({ destroyedFaces })
         forceToFrenzy() (via useEnergy callback)

  Vault contribution in HEIST mode:
    If heistState && score > 0:
      vaultContrib = Math.round(score * heistState.vaultSplit)
      personal = score - vaultContrib
      unbanked += personal (not full score)
      heistState.vault += vaultContrib
      dispatch VAULT_BETRAYAL_CHECK({ playerId, energy })
