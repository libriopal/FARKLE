VISUAL SPEC:
  LAYOUT: Single horizontal bar, full width of container.
  Two visual zones separated by a tick at 50% (FRENZY_THRESHOLD/MAX_ENERGY):

  [=====PRIME ZONE=====|=====FRENZY ZONE=====]
                       ^ tick mark at 50%

  PRIME fill (0–150): gradient left→right
    from: sky-600, to: blue-500
    When energy in this range: bar fills from left to tick

  FRENZY fill (150–300): gradient left→right
    from: orange-500, to: rose-600
    Both zones fill simultaneously once above threshold

  THRESHOLD TICK: amber-400 vertical line, 2px wide, absolute positioned

  STATUS TAGS (displayed below bar, flex row):
    LOW ENERGY (energy < 75):
      → amber-500 text-xs badge, pulsing opacity
    CRITICAL (energy < 25):
      → red-500 text-xs badge, fast pulse (0.3s interval)
    ANCHOR:
      → if isAnchor: emerald-400 "⚡ ANCHOR" badge
    CATCHUP:
      → if catchupActive: sky-300 "↑ CATCH-UP" badge

  PRIME/FRENZY MODE LABEL:
    Left of bar: "PRIME" text-sky-400 OR "FRENZY" text-rose-400
    Font: font-black text-xs uppercase tracking-widest
    Animate between labels with Motion AnimatePresence
    Exit: opacity → 0, y → -8
    Enter: opacity → 0 → 1, y → 8 → 0

  TRANSITIONS:
    PRIME→FRENZY: bar flashes white (opacity 0→0.8→0 over 300ms)
                  then fills to red/gold gradient
    FRENZY→PRIME: bar drains right-to-left over 500ms to blue

  COMPACT VARIANT (compact=true):
    No status tags
    No mode label
    Bar height: 6px (vs 14px normal)
    Threshold tick still shown

FILL CALCULATION:
  primeFill = Math.min(energy, FRENZY_THRESHOLD) / FRENZY_THRESHOLD * 50%
  frenzyFill = energy > FRENZY_THRESHOLD
    ? (energy - FRENZY_THRESHOLD) / FRENZY_THRESHOLD * 50%
    : 0%
  Total fill width = primeFill + frenzyFill

NUMERIC VALUE: displayed right of bar, font-black, color matches mode
