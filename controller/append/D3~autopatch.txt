ENCRYPTION ARCHITECTURE:
  Key Exchange: ECDH (P-256 curve)
  Message Encryption: AES-GCM (256-bit derived key)
  Each player generates an ECDH key pair on joining a room.
  Public keys are exchanged via WebSocket (plaintext — this is safe).
  Shared AES key derived via ECDH + HKDF.
  Each message uses a unique 96-bit IV.

KEY PAIR GENERATION (on mount):
  const keyPair = await crypto.subtle.generateKey(
    { name: 'ECDH', namedCurve: 'P-256' },
    true,
    ['deriveKey']
  );
  Store in useRef (never in state — avoid serialization).
  Export public key as JWK for sharing:
    crypto.subtle.exportKey('jwk', keyPair.publicKey)

SHARED KEY DERIVATION:
  When receiving another player's public key:
    import their JWK via crypto.subtle.importKey
    derive shared key via ECDH:
      crypto.subtle.deriveKey(
        { name: 'ECDH', public: theirPublicKey },
        localPrivateKey,
        { name: 'AES-GCM', length: 256 },
        false, ['encrypt', 'decrypt']
      )
    store in Map<senderId, CryptoKey>
  Use a single shared key for the room (all players derive same key
  by convention — in prod use a room key distribution mechanism).
  For simplicity: use first-joined player's derived key for all room messages.

ENCRYPT MESSAGE:
  async function encrypt(plaintext: string, key: CryptoKey):
    Promise<{ ciphertext: string; iv: string }>
    iv = crypto.getRandomValues(new Uint8Array(12))
    encrypted = await crypto.subtle.encrypt(
      { name: 'AES-GCM', iv },
      key,
      new TextEncoder().encode(plaintext)
    )
    return {
      ciphertext: btoa(String.fromCharCode(...new Uint8Array(encrypted))),
      iv: btoa(String.fromCharCode(...iv))
    }

DECRYPT MESSAGE:
  async function decrypt(ciphertext: string, iv: string, key: CryptoKey):
    Promise<string>
    ivBytes = Uint8Array.from(atob(iv), c => c.charCodeAt(0))
    ctBytes = Uint8Array.from(atob(ciphertext), c => c.charCodeAt(0))
    decrypted = await crypto.subtle.decrypt(
      { name: 'AES-GCM', iv: ivBytes },
      key,
      ctBytes
    )
    return new TextDecoder().decode(decrypted)

sendMessage(plaintext):
  1. Validate: plaintext.length <= CHAT_MAX_LENGTH
  2. Encrypt with room shared key
  3. Create ChatMessage with:
      senderArtSnippet: first 3 lines of localProfile.asciiArt
      encryptedContent, iv from encrypt()
  4. Call onSendRaw({ type: 'CHAT_MESSAGE', message: chatMsg })
  5. Add to local messages array (optimistic)

receiveEncryptedMessage(msg):
  1. Decrypt msg.encryptedContent using stored key for msg.senderId
  2. If decryption fails: show "[encrypted]" as content
  3. Add decrypted version to messages array
  4. Trim to CHAT_MAX_MESSAGES (remove oldest)
